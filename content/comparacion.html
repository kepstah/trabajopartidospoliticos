Comparación general

La síntesis compara agrupamientos amplios para ordenar el panorama general sin perder de vista que cada espacio tiene matices propios.

Rol del Estado

LLA / PRO: Mínimo / Facilitador

PJ / FIT-U: Central / Regulador

UCR / CC-ARI / HCF: Equilibrador / Institucional

Prioridad Económica

LLA / PRO: Mercado e Inversión

PJ / FIT-U: Consumo y Empleo

UCR / CC-ARI / HCF: Producción y Educación

Vínculo Externo

LLA / PRO: Occidente (EE.UU./Israel)

PJ / FIT-U: Multilateral / Regional

UCR / CC-ARI / HCF: Institucional / Diplomático

Coincidencias

Casi todos los partidos (salvo FIT-U) aceptan el sistema democrático republicano y la propiedad privada como base, aunque con distintos grados de regulación.

Diferencias

La principal brecha actual se da entre el "estatismo" (PJ/FIT-U) y el "liberalismo" (LLA/PRO), con el "institucionalismo" (UCR/CC/HCF) intentando mediar en el centro.
